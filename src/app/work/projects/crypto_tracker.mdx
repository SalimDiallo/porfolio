## Vue d'ensemble

Crypto Tracker est une application web moderne permettant de suivre en temps réel les prix des cryptomonnaies. Avec une interface élégante et des fonctionnalités avancées, elle offre aux utilisateurs une vue complète du marché crypto.

## Motivation du projet

Le marché des cryptomonnaies étant en constante évolution, les investisseurs ont besoin d'outils fiables pour suivre leurs actifs. Cette application répond à ce besoin avec une interface intuitive et des données en temps réel.

## Fonctionnalités principales

- **Prix en temps réel** : Mise à jour automatique des cours
- **Graphiques interactifs** : Visualisation de l'évolution des prix
- **Liste de surveillance** : Suivi des cryptos favorites
- **Statistiques détaillées** : Volume, capitalisation, variations
- **Comparaison** : Analyse comparative entre différentes cryptos

## Technologies utilisées

- **React** : Framework pour une interface réactive
- **Tailwind CSS** : Design moderne et responsive
- **CoinGecko API** : Données de marché fiables et actualisées

## Fonctionnalités techniques

- **Mise à jour en temps réel** : Polling automatique des données
- **Cache intelligent** : Réduction des appels API
- **Responsive design** : Adapté à tous les écrans
- **Performance optimisée** : Chargement rapide et fluide
- **Gestion d'état** : React Hooks pour une logique claire

## Interface utilisateur

L'application propose :
- Tableau de bord avec vue d'ensemble du marché
- Cartes détaillées pour chaque cryptomonnaie
- Graphiques interactifs avec différentes périodes
- Mode sombre pour le confort visuel
- Navigation intuitive et rapide

## API CoinGecko

Intégration complète avec :
- Prix en temps réel de milliers de cryptos
- Données historiques pour les graphiques
- Statistiques de marché globales
